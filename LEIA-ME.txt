Para instalar as dependências necessárias para utilização do programa basta utilizar o comando

pip install -r requirements.txt

no prompt de comando após se mover até o diretório onde o arquivo requirements.txt se encontra. Para isso utilize os comandos dir (listar os conteúdos de uma pasta) e 
cd (mudar de diretório). Exemplo (alguns itens omitidos por privacidade):
